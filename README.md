# Dynamic String Sales Project

This project is a C++ program that implements a custom dynamic string class and uses it to process and analyze sales data.

## Features
- Custom dynamic string class with memory management and operator overloading
- Stores and manages sale entries including:
  - Item name
  - Main category
  - Subcategory
  - Image
  - Link
  - Rating
  - Number of ratings
  - Discount price
  - Actual price
- Supports queries such as:
  - Displaying sale entry details
  - Calculating total number of sale entries
  - Outputting all sale links
  - Displaying unique ratings

## Technologies Used
- C++
- Object-Oriented Programming
- Data Structures
- Memory Management

## Purpose
This project was created as part of my Computer Science coursework at Kent State University to practice building custom data structures and managing real-world data.
